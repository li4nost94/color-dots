# Лоппи Поппи — Финансовый дашборд

Локальный сервис для визуализации выгрузок из 1С:Бухгалтерия.

## Быстрый старт

### 1. Положите XML-файлы в папку `data/`

```
data/
  2024_01.xml
  2024_02.xml
  ...
  2025_12_не_все_1с.xml
```

### 2. Запустите парсер

```bash
python3 scripts/parse_1c.py data/
```

Парсер:
- обработает все `.xml` файлы из папки
- создаст/обновит базу `scripts/loppipoppi.db`
- пропустит файлы, которые не изменились (по хешу)

### 3. Запустите сервер

```bash
python3 scripts/server.py
```

Откройте в браузере: **http://localhost:8080**

## Обновление данных

При получении новой выгрузки из 1С:
1. Положите файл в `data/`
2. Запустите `python3 scripts/parse_1c.py data/`
3. Обновите страницу в браузере

## Структура

```
loppipoppi-finance/
├── data/                  # XML-файлы из 1С
├── scripts/
│   ├── parse_1c.py       # Парсер XML → SQLite
│   ├── server.py         # API-сервер + статика
│   └── loppipoppi.db     # База данных (создаётся автоматически)
├── frontend/
│   └── index.html        # Дашборд
└── README.md
```

## Требования

- Python 3.8+
- Никаких внешних зависимостей! (только стандартная библиотека)

## Вкладки дашборда

- **Обзор** — метрики, графики движений, категории расходов
- **P&L** — отчёт о прибылях/убытках с конвертацией в USD
- **Зарплаты** — выплаты по ведомостям (зарплата + аванс + прочее)
- **Кадры** — действующие/уволенные сотрудники
- **ОС** — реестр основных средств
- **Валюты** — курсы EUR, USD, PLN за месяц
